HTML Beginning Tutorial

What is HTML?

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating web pages.

Creating an HTML Document

To create an HTML document, you need to use tags to define the structure and content of the page.

Formatting Tags

Formatting tags are used to format the text and layout of the page. Examples include bold, italic, and underline.

Viewing an HTML Page

To view an HTML page, you can use a web browser such as Chrome, Firefox, or Safari.

For more information on creating HTML pages, work through this HTML Tutorial.
